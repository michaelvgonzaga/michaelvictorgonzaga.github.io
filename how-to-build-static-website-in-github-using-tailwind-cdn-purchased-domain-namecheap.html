How to Build a Static Website with Tailwind in Under 15 Minutes

By Michael Victor Gonzaga — August 2025

Prerequisites

  • GitHub account
  • Text editor or GitHub web editor
  • Optional: a domain from Namecheap

Step 1 — Create GitHub Pages Repo

Go to GitHub → New repository → name it <username>.github.io → set to Public.

Step 2 — Add Homepage

Create index.html with Tailwind via CDN:

<script src="https://cdn.tailwindcss.com"></script>

Step 3 — Enable GitHub Pages

Repo → Settings → Pages → Source: main / (root) → Save.

Step 4 — Connect Namecheap Domain

In Namecheap → Advanced DNS, add:

A @ 185.199.108.153
A @ 185.199.109.153
A @ 185.199.110.153
A @ 185.199.111.153
CNAME www <username>.github.io

Step 5 — Enforce HTTPS

Wait for SSL to provision, then tick Enforce HTTPS.

That’s it — you now have a fast, responsive static website online in less than 15 minutes!
